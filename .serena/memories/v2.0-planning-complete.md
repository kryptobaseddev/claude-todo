## Key Decisions

### Architecture
- **Storage**: SQLite primary + JSONL for git sync
- **Dependencies**: 4 types (blocks, related, discovered_from, parent-child)
- **IDs**: Hierarchical (T001.1.1) with optional hash suffix

### Phase Plan
1. **Foundation**: SQLite, enhanced deps, ready detection, hierarchical IDs
2. **Distribution**: Git sync, JSONL, merge driver, protected branches
3. **Intelligence**: Compaction, agent coordination, semantic search
4. **Visualization**: TUI, web dashboard, Kanban
5. **Enterprise**: Multi-project, teams, plugins

### Key Features from beads to Adopt
- SQLite caching (~100ms queries vs file reads)
- 4 dependency types with blocking semantics
- Ready work detection algorithm
- Memory decay/compaction for old tasks
- Git-native distribution with merge drivers
- Multi-agent coordination

### claude-todo Strengths to Preserve
- 4-layer anti-hallucination validation
- Session management & focus mode
- Phase management system
- Comprehensive backup rotation
- TodoWrite integration
